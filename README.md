### Features

- [x] Vue.js 2+
- [x] Element UI
- [x] WYSIWYG Editor
- [x] Markdown Editor
- [x] Easy to deploy

### How to Start

```bash
# clone project
git clone https://github.com/geekpark/gpk_admin.git

cd src
# modify config.js
cp config.example.js config.js
cd ..
yarn install
# start
npm run dev
```
